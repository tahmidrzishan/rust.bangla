রাস্ট ভ্যারিয়েবলের mutability

বাই ডিফল্ট রাস্টে সব ভ্যারিয়েবল immutable. তার মানে, এগুলার মান একবার ঠিক করে দেয়ার পর আর চেঞ্জ করা যায় না। যদি আমরা এই immutable ভ্যারিয়েবলগুলোকে mutable করতে চাই, তাহলে আমাদের একটা স্পেশাল কিওয়ার্ড ইউজ করতে হয়। সেটা হলো mut.
fn main() {
    let age: i8 = 123;
    println!("amar boyosh {} bochor", age);
}
এই কোডটা রান করলে আমরা স্বাভাবিক এরকম একটা রেজাল্ট পাবোঃ
$ cargo run
Compiling hello v0.1.0 (/home/zishan/Desktop/hello)
Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.32s
Running `target/debug/hello`
amar boyosh 123 bochor
ভেজালটা হবে তখন, যখন আমরা কোন ভ্যারিয়েবলের মান পরিবর্তন করতে যাবো। এবার এই কোডটা রান করে দেখেন।
fn main() {
    let age: i8 = 123;
    println!("amar boyosh {} bochor", age);
    age = 123;
    println!("amar boyosh ekhon {} bochor", age);
}
এইরকম বিদঘুটে একটা এরর আসছে, তাই না?
$ cargo run
Compiling hello v0.1.0 (/home/zishan/Desktop/hello)
error[E0384]: cannot assign twice to immutable variable `age`
 --> src/main.rs:4:5
   |
2  |    let age: i8 = 123;
   |        ---
   |        |
   |        first assignment to `age`
   |        help: consider making this binding mutable: `mut age`
3  |    println!("amar boyosh {} bochor", age);
4  |    age = 123;
   |    ^^^^^^^^^ cannot assign twice to immutable variable

For more information about this error, try `rustc --explain E0384`.
error: could not compile `hello` (bin "hello") due to 1 previous error
আসলে বিদঘুটে না। এই এরর টা আসছে আপনাকে হেল্প করার জন্য। রাস্ট এর ভ্যারিয়েবলগুলো যেহেতু বাই ডিফল্ট immutable, মানে, মান পরিবর্তন করা যায় না, তাই রাস্ট আপনাকে এই এররটা দিচ্ছে। এররটাতে দেখেন রাস্ট আপনাকে বলে দিচ্ছে কী সমস্যা হয়েছে, আর সেটা কিভাবে ঠিক করতে হবে। এররটা হলো,
error[E0384]: cannot assign twice to immutable variable `age`
রাস্ট এখানে আপনাকে বলছে যে, আপনি একটা imuutable variable এ একাধিকবার মান বসাতে পারবেন না। এরপরে রাস্ট আপনাকে হেল্পও করতে চাইছে।
help: consider making this binding mutable: `mut age`
এখন, এটা কিভাবে করবেন? আসলে অনেক সহজ। জাস্ট ভারিয়েবলের নামের আগে mut keyword টা দিয়ে দিতে হবে। তাহলেই আপনার ভ্যারিয়েবলটার আপনি যতবার ইচ্ছা মান চেঞ্জ করতে পারবেন।
let mut age: i8 = 123;
এইভাবে। এবার নিচের মত কোডটা লিখে রান করে দেখেন।
fn main() {
    let mut age: i8 = 123;
    println!("amar boyosh {} bochor", age);
    age = 123;
    println!("amar boyosh ekhon {} bochor", age);
}
$ cargo run
Compiling hello v0.1.0 (/home/zishan/Desktop/hello)
Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.31s
Running `target/debug/hello`
amar boyosh 123 bochor
amar boyosh ekhon 123 bochor
এইবার সব ঠিক আছে। কারণ আমরা ভ্যারিয়েবলটাকে mutable করে নিয়েছি।

উপসংহার

রাস্টের সব ভ্যারিয়েবলই immutable. অর্থাৎ মান পরিবর্তন করা যায় না। মান পরিবর্তন করা যায় এমন ভ্যারিয়েবল চাইলে  mut keyword টা ইউজ করতে হয়।
