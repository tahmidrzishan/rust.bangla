স্ট্রিং টাইপ

ভ্যারিয়েবল প্রিন্ট করা শেখার আগে আমরা আরেকটা ডাটা টাইপ শিখে নিবো, সেটা হলো স্ট্রিং। নরমালি আমরা কোন টেক্সট টাইপের ডাটা রাখার জন্য এই টাইপটা ইউজ করি।
let name : &str = "Zishan";
এভাবে একটা স্ট্রিং ভ্যারিয়েবল ডিক্লেয়ার করার পরে আমরা এটা ঈঊজ করতে পারবো। এবার চলেন, আমরা কয়েকটা ভ্যারিয়েবল প্রিন্ট করি।

ভ্যারিয়েবল প্রিন্ট

fn main() {
    let name : &str = "Zishan";
    let age: i8 = 123;
    println!("amar nam holo {}", name);
    println!("amar boyosh matro {} bochor", age);
}
এবার যদি আমরা cargo run কমান্ড দিয়ে কোডটা রান করি তাহলে নিচের মত রেজাল্ট দেখতে পাবো।
$ cargo run
Compiling hello v0.1.0 (/home/zishan/Desktop/hello)
Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.84s
Running `target/debug/hello`
amar nam holo Zishan
amar boyosh matro 123 bochor

অ্যানালাইসিস

let name : &str = "Zishan";
let age: i8 = 123;
এখানে let keyword টা দিয়ে একটা নতুন ভ্যারিয়েবল তৈরি করা হয়। "let" মানে "হইতে দাও"। let name : &str = "Zishan"; মানে হইলো, name নামের একটা ভ্যারিয়েবল হইতে দাও যার টাইপ হবে স্ট্রিং, আর মান হবে, "Zishan".
আর let age: i8 = 123; এর মানে হইলো age নামের একটা ভ্যারিয়েবল হইতে দাও যার টাইপ হবে 8 bit এর integer আর মান হবে 123.
এবার আমরা চাইলে আমাদের এই ভ্যারিয়েবলগুলা প্রিন্ট করতে পারি।
println!("amar nam holo {}", name);
println!("amar boyosh matro {} bochor", age);
আমরা প্রিন্ট লাইন দিয়ে ডাবল কোট এর ভিতরে একটা স্ট্রিং লিখে দিয়েছি। এটা প্রিন্ট হবে। এই জিনিসটা আমরা আমাদের হ্যালো ওয়ার্ল্ড প্রোগ্রামেই শিখে এসেছি। কিন্তু {} এই জিনিসগুলা আবার কী? আর এগুলা প্রিন্ট হইলো না কেন? মূলত এগুলা হইলো আমাদের ভ্যারিয়েবল এর জন্য placeholder. খেয়াল করে দেখেন আমরা প্রিন্ট এর ভিতরে স্ট্রিং টা দেয়ার পরে কমা দিয়ে আমাদের ভ্যারিয়েবলের নামটা দিয়ে দিয়েছি। আমরা এখানে বলে দিচ্ছি, আমাদের স্ট্রিং এর ভেতরে আমরা যে প্লেসহোল্ডারটা রেখেছি, সেখানে কোন ভ্যারিয়েবলটা বসবে। রাস্ট আমাদের প্রোগ্রামটা রান করার সময় {} এর জায়গায় আমাদের ভ্যারিয়েবলের মানটা বসিয়ে দিবে।

উপসংহার

  • let keyword টা দিয়ে আমরা কোন ভ্যারিয়েবল ডিক্লেয়ার করতে পারি। এসময় আমাদের কোলন দিয়ে ভ্যারিয়েবলের টাইপটা লিখে দিতে হয়।
  • কোন ভ্যারিয়েবল প্রিন্ট করার সময় আমরা curly braces {} দিয়ে ভ্যারিয়েবলের জন্য জায়গা রেখে দেই। পরে কমা দিয়ে ভ্যারিয়েবলের নাম লিখে দিলে রাস্ট কম্পাইলার আমাদের জন্য সেটা রিপ্লেস করে দেয়।
