গাণিতিক অপারেশন

রাস্টে বেসিক পাঁচটা গাণিতিক অপারেশন করা যায়। যোগ, বিয়োগ, গুণ, ভাগ, আর ভাগশেষ নির্ণয়। নিচে ডিক্লেয়ার করা দুইটা ভ্যারিয়েবলের মধ্যে আমরা গাণিতিক অপারেশনগুলো করে দেখবো।
let num1: i8 = 15;
let num2: i8 = 4;
এবার আমরা এই ভ্যারিয়েবলগুলোর যোগফল, বিয়োগফল, গুণফল, ভাগফল, আর ভাগশেষ প্রিন্ট করতে পারি।
println!("num1 + num2 = {}", num1 + num2);
println!("num1 - num2 = {}", num1 - num2);
println!("num1 * num2 = {}", num1 * num2);
println!("num1 / num2 = {}", num1 / num2);
println!("remainder = {}", num1 % num2);
এটা রান করলে রেজাল্টটা হবে কিছুটা এইরকমঃ
$ cargo run
Compiling hello v0.1.0 (/home/zishan/Desktop/hello)
Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.31s
Running `target/debug/hello`
num1 + num2 = 19
num1 - num2 = 11
num1 * num2 = 60
num1 / num2 = 3
remainder = 3
আমরা চাইলে এই রেজাল্টগুলোকে কোন ভ্যারিয়েবলেও সেভ করে রাখতে পারি।
let add = num1 + num2;
let sub = num1 - num2;
let multi = num1 * num2;
let dev = num1 / num2;
let rem = num1 % num2;
এরপরে জাস্ট এগুলো প্রিন্ট করে দিবো।
println!("{}", add);
println!("{}", sub);
println!("{}", multi);
println!("{}", dev);
println!("{}", rem);
এবার cargo run করলে আপনি নিচের মত রেজাল্ট দেখতে পারবেন ইনশাআল্লাহ।
$ cargo run
Compiling hello v0.1.0 (/home/zishan/Desktop/hello)
Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.34s
Running `target/debug/hello`
